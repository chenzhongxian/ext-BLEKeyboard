# Mind+ 扩展库：ESP32 蓝牙键盘

![](./arduinoC/_images/featured.png)

## 简介

本扩展库仅限在 Mind+ 软件中使用，为 ESP32（掌控板）设计，可以将掌控板模拟成蓝牙键盘，支持媒体键，可以在 Mind+ 用户库中进行导入。
